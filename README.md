# Brain Games
Command-line games.

## Installation
Clone the repository:  `https://github.com/SpawnLeon/frontend-project-lvl1`

Then type:
```
make install
npm link
```

## Starting games
Type:
* `brain-even` - guess the parity of a number;
* `brain-calc` - calculate a math expression;
